Dropdown

Element

O usuário deve entender como funciona uma lista de opções pré definidas, seja para um campo para entrada de dados ou em um menu.

Checklist

0 de itens
Marque na checklist abaixo as práticas que você está seguindo
  • Título dos itens
    Use títulos claros e úteis para a pessoa usuária em cada um dos itens da lista
    Evite o uso de termos legais, estrangeiros e técnicos no título de cada uma das opções da lista
    Evite usar palavras difíceis para o seu público
  • Opções da lista
    Use um universo de opções que contemplem os desejos da pessoa usuária
  • Placeholder
    Use placeholders úteis para a pessoa usuária
    Evite o uso de abreviações
    Evite descrever o campo por meio de um placeholder já que eles desaparecem assim que o usuário começa a digitar
    Exiba exemplos de preenchimento do campo no placeholder
  • Opcional
  • Desejos da pessoa usuária
    Considere incluir a opção de 'todos' para campos de busca ou filtro com uma lista de opções
