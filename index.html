Checklist da Linguagem Simples

Conjunto de melhores práticas para facilitar o entendimento dos usuários de portais de transparência.

O que é Linguagem Simples?

É uma técnica para melhoria da comunicação que faz com que a informação seja compreendida de forma mais rápida, fácil e precisa pelo público. Alguns exemplos de práticas desse tipo de comunicação são: o uso de frases curtas e diretas, a organização lógica das informações e o uso de ilustrações e exemplos. Sendo assim, podemos perceber que a linguagem simples atua não só no conteúdo textual, mas também na estrutura e no design da informação.

Por que usar checklists?

Utilizando os padrões

1. Escolha um elemento do site que você deseja implementar as práticas de linguagem simples. Esse elemento pode ser, por exemplo, um botão que as pessoas que acessam o seu site estão com dificuldade de entender a funcionalidade que está por trás dele.
2. Acesse a checklist correspondente ao tipo de elemento de interface que você deseja tornar mais simples na página de elementos. No caso do nosso exemplo, a categoria botão.
3. Aplique todas as recomendações da lista ao elemento do seu site.
4. Repita esse processo para todos os elementos do trecho que você deseja tornar mais simples para o usuário.
Dica: Marque o checkbox de cada item conforme você for seguindo as recomendações contidas nele. Isso serve para monitorar seu progresso na checklist.
